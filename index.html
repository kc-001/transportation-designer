Why TransportationDesigner

Built for how DOT
designers actually work

These tools aren't adapted from general-purpose software. They're purpose-built around FDOT standards, project workflows, and the calculations designers run every day.

FDOT Standards Aligned

Constraint limits, lane widths, and design criteria match current FDOT Design Standards and PPM. No translating between tools and specifications.

No IT Approval Required

Runs entirely in the browser. Nothing to install, no admin rights needed, no VPN required. Share a link and it works.

Open, Inspectable Logic

Every calculation is transparent. Engineers can see the formulas, verify the math, and trust the output — or catch an error before it reaches a set of plans.

Built for Project Teams

Share a configuration URL with a colleague or reviewer. Everyone sees the same numbers without emailing spreadsheets back and forth.

How It Works

From browser to
design decision in minutes

1

Open a tool

Click any tool from the directory. It loads instantly in your browser — desktop or tablet.

2

Enter your parameters

Input lane widths, design speeds, or curve geometry. Results update live as you adjust.

3

Check constraints

The tool flags anything that falls outside FDOT standard ranges — MSE limits, sight distances, K-values.

4

Export or share

Download a CSV or JSON summary, print to PDF, or copy the URL to share your configuration with a reviewer.
