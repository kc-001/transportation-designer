Governing Equations
Minimum Spiral Length — Comfort Criterion
Ls = 3.15 × V³ / (R × C)
V = Design speed (mph) · R = Radius (ft) · C = 1.0 ft/s³
Minimum Spiral Length — Runoff Criterion
Ls = (w × n × |ed|) / Δ
w = Lane width · n = Number of lanes · ed = emax − eNC · Δ = max relative gradient (~0.45%/ft for 60mph)
Spiral Angle at SC
θs = Ls / (2R) × (180/π)
Central angle from TS to SC
Spiral Geometry (power series)
Xs = Ls[1 − θs²/10 + θs&sup4;/216]
Ys = Ls[θs/3 − θs³/42]
In radians. Shift p = Ys − R(1−cosθs)
Worked Example
Problem: V = 60 mph, R = 1500 ft, emax = 0.08, eNC = 0.02, 2 lanes @ 12 ft

Step 1 — Comfort criterion:
Ls = 3.15 × 60³ / (1500 × 1.0) = 3.15 × 216000 / 1500 = 453.6 ftStep 2 — Runoff criterion:
Lr = (12 × 2 × |0.08 − 0.02|) / 0.0045 = 1.44 / 0.0045 = 320 ftResult: Ls = max(453.6, 320, 50) = 453.6 ft — comfort governs

FDOT rounds to next 25 ft: use 475 ft
