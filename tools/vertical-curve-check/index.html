Governing Equations
Crest — Case 1: S ≤ L (sight distance within curve)
L = A × S² / [100 × (√2h1 + √2h2)²]
A = |g2−g1| (%) · S = SSD (ft) · h1 = 3.5 ft · h2 = 2.0 ft (object)
Crest — Case 2: S > L (sight line extends beyond curve)
L = 2S − [100 × (√2h1 + √2h2)²] / A
Use when curve is shorter than sight distance — conservative result
Sag — Headlight Criterion (S ≤ L)
L = A × S² / (400 + 3.5S)
Governs nighttime conditions based on 2° upward headlight beam and 0.6 m object height
Sag — Overhead Clearance (check separately)
L = A × S² / (800 × C)
C = structure clearance minus driver/object heights
K-Value
K = L / A
Rate of grade change per percent — used directly in plan check
