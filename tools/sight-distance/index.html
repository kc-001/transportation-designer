Governing Equations
Stopping Sight Distance — AASHTO Eq. 3-1
SSD = 1.47Vt + V² / [30(a/g ± G)]
V = Speed (mph) · t = PRT (sec) · a = decel (ft/s²) · G = grade (decimal)
Use + G for downgrades (increases SSD), − G for upgrades (decreases SSD)
AASHTO table values assume level grade (G = 0)
Passing Sight Distance — AASHTO §3.3
PSD = d₁ + d₂ + d₃ + d₄
d₁ = Perception + acceleration distance
d₂ = Distance traveled in opposing lane
d₃ = Clearance gap to opposing vehicle (~200 ft)
d₄ = Opposing vehicle distance during d₂
Note
Refer to AASHTO Green Book Table 3-4 for exact tabulated PSD values by speed
The calculation here uses the AASHTO component method with typical passing times of 10–14 sec
Decision Sight Distance — AASHTO §3.3
DSD = 1.47V × (t_r + t_m)
t_r = Extended perception-reaction time (3.0–4.5 sec for complex scenarios)
t_m = Avoidance maneuver time (varies 5–11 sec by maneuver type A–E)
DSD >> SSD — used at complex decision locations: interchanges, intersections, decision points
Maneuver Types
A: Stop rural · B: Stop urban · C–E: Speed/path/direction change
Maneuvers C, D, E require significantly more sight distance than stopping maneuvers
