Governing Equations
K-Value
K = L / |A|
L = Curve length (ft) · A = |g2 − g1| (%)
BVC & EVC Stationing
BVC = PVI − L/2    EVC = PVI + L/2
Station arithmetic
BVC & EVC Elevation
BVC elev = PVI elev − (g1/100) × (L/2)
EVC elev = PVI elev + (g2/100) × (L/2)
Grade applied from PVI
Elevation at Distance x from BVC
y = BVC_elev + (g1/100)×x + (A/(200L))×x²
Parabolic equation
High/Low Point (zero-slope)
xHL = −g1 × L / A   (0 < x < L)
Only valid when high/low is within curve limits
